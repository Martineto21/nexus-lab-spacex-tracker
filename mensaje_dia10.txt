*🧠 Instrucciones para Nexus Platform:*

*1.* Ve a tu Workspace → proyecto SpaceX Tracker
*2.* Abre nueva pestaña y escribe:

```
"Configura el proyecto para release de producción.
Necesito:
- ProGuard rules para optimización
- Signing config para firmar el APK  
- Script de build automático
- Guía paso a paso para Play Store
- Optimizaciones de rendimiento"
```

*3.* El agente generará todos los archivos de configuración

*4.* Para generar el build final, escribe:

```
"Ejecuta el build de release.
Genera tanto APK como Android App Bundle.
Optimiza el código y elimina dependencias innecesarias."
```

*🔥 Lo que vas a obtener:*
• *app-release.apk* - Para instalar y probar
• *app-release.aab* - Para subir a Play Store  
• *Guía completa* - Todos los pasos para publicar
• *Assets gráficos* - Iconos y screenshots
